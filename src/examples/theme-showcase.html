Características del Sistema de Tema

Detección Automática

Detecta automáticamente la preferencia del sistema y aplica el tema correspondiente.

Persistencia

Guarda la preferencia del usuario en localStorage para mantener el tema entre sesiones.

Transiciones Suaves

Transiciones animadas entre temas para una experiencia fluida y profesional.

Sistema de Tokens

Utiliza tokens UBITS oficiales que se adaptan automáticamente al tema seleccionado.

Integración de Componentes

Todos los componentes UBITS se adaptan automáticamente al tema sin configuración adicional.

Integración Vue.js

Hook useTheme() para integración reactiva con Vue.js 3 y TypeScript.
