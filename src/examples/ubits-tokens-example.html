💻 Código de Ejemplo

Cómo usar los tokens UBITS en tu código CSS y JavaScript. 

/* CSS - Usar tokens UBITS */
.my-component {
  background-color: var(--ubits-bg-1);
  color: var(--ubits-fg-1-high);
  border: 1px solid var(--ubits-border-1);
}

.my-button {
  background-color: var(--ubits-button-primary-bg-default);
  color: var(--ubits-btn-primary-fg);
}

.my-button:hover {
  background-color: var(--ubits-button-primary-hover);
}

/* JavaScript - Obtener tokens UBITS */
const brandColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--ubits-accent-brand');

const backgroundColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--ubits-bg-1');
